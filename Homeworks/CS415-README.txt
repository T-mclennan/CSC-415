CS 415 is the operating systems class offered at San Francisco State University.
The class uses C with the POSIX API for all assignments. Topics covered by the course include:

 - Operating system design / resource allocation.
 - Scheduling algorithms
 - Multithreading, multiprocessing, and synchronization techniques.
 - Memory management systems
 - Virtual memory, Paging algorithms, disk algorithms.  

All assignments are submitted with a written analysis.
